## JPA CLI v1.0 🚀

Kami dengan bangga mengumumkan rilis JPA CLI v1.0 — tool CLI untuk mengelola konfigurasi AI Agent Anda.

### Fitur Utama

1. **Auto-Detect Workspace** — Otomatis mendeteksi editor (Cursor, VS Code, Claude Code, Windsurf, Zed)
2. **Sync SKILL.MD** — Browse dan install 50+ skill files dari library
3. **Sync DESIGN.MD** — Template desain untuk berbagai framework
4. **MCP Management** — Browse, install, dan konfigurasi MCP servers
5. **1-Click Cloner** — Install preset langsung ke workspace

### Instalasi

```bash
npm install -g @jagopakaiai/jpa-cli
```

### Apa Selanjutnya?

- Custom skill upload ke community
- Team workspace sync
- VS Code extension
