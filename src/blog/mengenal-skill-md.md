## Apa itu SKILL.MD?

SKILL.MD adalah file direktif yang memberitahu AI Agent (seperti Cursor, Claude Code, atau Copilot) tentang kemampuan, aturan, dan instruksi spesifik yang harus diikuti saat bekerja dalam project Anda.

## Mengapa Penting?

Tanpa SKILL.MD, AI Agent bekerja dengan pengetahuan umum. Dengan SKILL.MD, AI Agent Anda:

- **Memahami konteks project** — tahu stack teknologi, konvensi kode, dan arsitektur
- **Mengikuti aturan tim** — konsisten dengan coding standard dan best practices
- **Lebih produktif** — tidak perlu menjelaskan ulang preferensi di setiap prompt

## Cara Menggunakan

```bash
# Install JPA CLI
npm install -g @jagopakaiai/jpa-cli

# Browse skill library
jpa-cli skills

# Install skill ke workspace
jpa-cli sync next-js-expert
```

Mulai gunakan SKILL.MD hari ini dan rasakan perbedaannya!
