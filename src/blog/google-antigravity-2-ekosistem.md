## Antigravity: Agent-First Development

Google Antigravity awalnya diluncurkan November 2025 sebagai VS Code-based IDE. Namun di Google I/O 2026, platform ini bertransformasi menjadi ekosistem developer tools yang komprehensif.

### Komponen Antigravity 2.0

1. **Antigravity Standalone App** — Desktop app untuk macOS, Windows, Linux sebagai command center untuk workflow agent paralel
2. **Antigravity IDE** — IDE agentic original dengan integrasi codebase yang mendalam
3. **Antigravity CLI** — Tool terminal untuk developer yang prefer keyboard-centric workflow
4. **Antigravity SDK** — Integrasi programmatic dari harness orkestrasi agent ke infrastruktur custom

### Dual-Interface Design

Antigravity memiliki desain interface ganda yang unik:

- **Editor View** — Untuk hands-on coding tradisional
- **Manager View** — Untuk orkestrasi agent otonom

### Kemampuan Otonom

Agent Antigravity bisa secara otonom menavigasi editor, terminal, dan browser untuk melakukan task end-to-end: menulis kode, menjalankan test, dan memverifikasi UI secara visual.

### Multi-Model Support

Platform ini dirancang untuk memanfaatkan model Gemini (Gemini 3.5 Flash) tetapi juga mendukung integrasi dengan model lain, memberikan fleksibilitas dalam cara agent berpikir dan mengeksekusi.

### Artifact System

Agent menghasilkan "Artifacts" — output terstruktur termasuk plans, test results, dan visual proof of work — untuk memberikan transparansi atas progress mereka.

> Antigravity 2.0 bukan sekadar IDE upgrade — ini adalah platform untuk era "vibe coding".
