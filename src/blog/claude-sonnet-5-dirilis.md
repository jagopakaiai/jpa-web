## Claude Sonnet 5: Model Baru untuk Era Agentic

Anthropic resmi merilis Claude Sonnet 5 sebagai model terbaru yang dirancang khusus untuk performa tinggi dalam workflow coding agentic.

### Apa yang Baru?

Claude Sonnet 5 membawa peningkatan signifikan dalam beberapa area:

- **Reasoning yang lebih dalam** — Kemampuan multi-step reasoning yang lebih konsisten
- **Context window yang besar** — Memungkinkan agent membaca dan memahami codebase berskala besar
- **Tool-calling yang lebih akurat** — Mengurangi error saat berinteraksi dengan terminal, file system, dan API
- **Speed optimization** — Respons lebih cepat untuk iterasi development yang intensif
