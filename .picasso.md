# .picasso.md — CIM Design Direction

## Direction: Nothing Instrument Panel

A single-viewport bento dashboard inspired by Nothing's industrial design language. OLED black, monochrome hierarchy, dot-matrix display typography, mechanical precision. One accent: Nothing Red on the CIM World tile.

## Design Dials
- DESIGN_VARIANCE: 7 (avant-garde bento grid, asymmetric tile sizes)
- MOTION_INTENSITY: 3 (hover states only, no scroll animations — single viewport)
- VISUAL_DENSITY: 8 (dense bento grid, data-heavy stats, compact tiles)

## Typography
- Display: Doto (variable dot-matrix, 48-96px, hero/stats only)
- Body: Space Grotesk (300/400/500, headings/descriptions)
- Data/Labels: Space Mono (400, ALL CAPS, 0.08em tracking, 11px)
- Max 3 sizes per screen, 2 weights

## Color Tokens (Dark Mode Default)
- Background: #000000 (OLED black)
- Surface: #111111
- Surface Raised: #1A1A1A
- Border: #222222 / #333333
- Text: #FFFFFF (display) / #E8E8E8 (primary) / #999999 (secondary) / #666666 (disabled)
- Accent: #D71921 (Nothing Red — CIM World tile only)
- Interactive: #5B9BF6 (links)

## Color Tokens (Light Mode)
- Background: #F5F5F5
- Surface: #FFFFFF
- Text: #000000 (display) / #1A1A1A (primary) / #666666 (secondary) / #999999 (disabled)

## Layout
- Single viewport (100vh, no scroll on desktop)
- CSS Grid bento: 12 columns × 6 rows
- 8px tile gaps, 12px border-radius
- Mobile: flex column, scrollable
- Tablet: 6-column grid, scrollable

## Identity
- The ONE surprise: CIM World tile with red accent border + dot-grid background
- Tertiary labels feel like instrument panels (Space Mono, ALL CAPS, tiny)
- Stats in Doto display font — data IS the visual
- No shadows, no gradients, no illustrations
- Flat surfaces separated by borders only

## Rejections
- No shadows or blur
- No gradients
- No purple/indigo/violet anything
- No centered vertical layout
- No uniform card grids (tiles are deliberately different sizes)
- No hover translate-y or shadow-lift effects
- No skeleton loading
- No filled icons or emoji
